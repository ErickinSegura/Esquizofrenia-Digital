### Para una variable

1) x - 0 = 0
2) x - 1 = x
3) x * x = x
4) x * x' = 0
5) x + 0 = x
6) x + 1 = 1
7) x + x = x
8) x + x' = 1

### Para múltiples variables

9) x + y = y + x
10) x * y = y * x
11) x + (y + z) = (x + y) + z = x + y + z
12) x(yz) = (xy)z = xyz
13) x(y + z) = xy + xz
14) (w + x)(y + z) = wy + xy + wz +xz
15) x + xy = x
16) x + x'y = x + y
17) x' + xy = x'y'
18) (x + y)' = x'y'
19) (xy)' = x' + y'
