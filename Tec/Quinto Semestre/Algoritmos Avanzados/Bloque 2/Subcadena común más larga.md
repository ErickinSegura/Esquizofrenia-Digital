# Pseudocódigo
```
LCS(T1,T2):
	n = |T1|, m = |T2|
	dp = matrix[0...n][0...m]
	for i = 0 to n
		dp[i,0]=0
	for i = 0 to m
		dp[0,i]=0
	max=0
	for i=0 to n
		for j=0 to m	
			if T1[i] = T2[j]
				dp[i,j] = dp[i-1,j-1]+1
			else
				dp[i,j] = 0
			max = (dp[i,j] > max) ? dp[i,j] : max
	return max
```
